## 制作の背景
自分の所属するゲームのコミニティの待ち時間(対戦相手検索中など)を有効活用出来たらいいなと思い、手軽に遊べるクイズアプリを作りました。

## 使用方法
上記のアドレスをクリックしたら画面中央の出題開始をクリックするとクイズが始まります。
